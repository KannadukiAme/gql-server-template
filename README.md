# gql-server-template

这是graphql server的一个模板，它使用以下技术栈。

- graphql
- type-graphql
- mikro-orm
- mongodb
- koa
- typescript
- docker
- docker-compose

## How to use it

cd ./mongo

start mongodb docker container

```bash
docker-compose up -d
```

start graphql server

```bash
yarn # install dependence
yarn dev # start server
```
